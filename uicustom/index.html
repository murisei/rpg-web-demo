Quick Templates

By clicking on Apply, a Quick Template overrides all individualization that you have already done. There's no undo button.

Touch UI

This section is only available in RPG Maker MZ.
  • When Touch UI is on, the player can still decide to disable it in the Game's Options menu.
  • When Touch UI is off, then "Show Touch UI" is also removed from the Game's Option menu.
  • When set to "When on mobile" then this Plugin calls an RPG Maker core code function to check whether the player is using a mobile device. This Plugin does not guarantee this check's reliability.

Tip: You may want to enable this checkbox when you selected "Show Touch UI when on mobile" so you don't end up with 2 slightly different window sizes.
